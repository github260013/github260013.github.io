gitbash命令

git init

这个命令将在当前目录中初始化一个新的 Git 仓库。在执行这个命令后，Git 会在当前目录下创建一个名为.git的子目录，用于存储仓库的版本历史和配置信息。

git commit -m "提交说明"

这个命令用于将暂存区的文件提交到本地仓库，并附上一条提交说明。例如，执行git commit -m "Fix a bug"会将暂存区的改动提交到本地仓库，并将提交说明记录为"Fix a bug"。

git push

当你想要将本地仓库的改动推送到远程仓库时，就可以使用这个命令。例如，执行git push origin master会将当前分支（通常是master分支）的改动推送到名为origin的远程仓库。

git clone

使用这个命令可以克隆一个远程仓库到本地。例如，执行git clone https://github.com/username/repository.git会将远程仓库克隆到当前目录下的一个名为repository的文件夹中。
